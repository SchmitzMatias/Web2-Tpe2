# Lify Rest API documentation

## Categories

> GET /api/categories

**Query Parameters**

| Params                    | Type          | Required | Description                                         |
|:-----------------------------|:--------------|:---------|:----------------------------------------------------|
| `order`                      | string        | No      | asc or desc.                                 |
| `page`                      | string        | No      | Current page, default no pagination.                                 |
| `limit`                      | string        | No      | No of categories per page.                                 |
